Data:

Contains all datasets necessary to generate figures 1-14 and analysis on final report.

Datasets are as follows:

EPA_ca_no2_2020_raw_data.csv


The EPA NO2 concentration in 2020 was downloaded from EPA official website (https://www.epa.gov/outdoor-air-quality-data/download-daily-data, choose NO2 and California to download.) This data set was used to generate figures 1-3.

fire_archive_J1V-C2_689564.csv


The fire activity in 2020 was downloaded from NASA FIRMS (Fire Information for Resource Management System) offical website (https://firms.modaps.eosdis.nasa.gov/). This data set was used to generate figure 13-14.
